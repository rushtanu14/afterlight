# Afterlight Vision Presentation

## Slide 1: The Future Has a Memory Problem

Every disaster teaches people what failed. Then the lesson disappears.

Afterlight preserves those failures as replayable decisions.

## Slide 2: Not Another Alert Dashboard

Existing tools show where the fire is, where the road is, or what the agency said.

Afterlight asks: when did the failure chain begin, and what should this household do before it repeats?

## Slide 3: The Demonstration

Enter a location.

Select a nearby historical fire.

Replay the timestamp chain.

Watch the detector mark the first leave-now threshold when hazard movement, route stress, and official-source confidence stack together.

## Slide 4: The Technical Core

The prototype contains a tested detector:

- route closure risk
- backup overload risk
- household mobility buffer
- recommended route
- evidence trail
- crisis cards

This is deliberately narrow. It is auditable instead of magical.

## Slide 5: The Moonshot

The long-term system is civic memory infrastructure.

Every wildfire, flood, storm, blackout, and heat event should teach the next household what to do earlier.

The future is not just better alerts.

The future is memory that operates.
